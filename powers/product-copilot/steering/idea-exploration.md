# 创意探索工作流

从模糊想法到清晰概念的对话引导。

## 第一步：捕捉原始想法

### 1.1 让用户自由表达

不要打断，让用户把想法说完。记录：
- 关键词
- 情绪（兴奋点、痛点）
- 隐含的假设

### 1.2 复述确认

```
"让我确认一下我的理解：
你想做的是 _______________
主要是为了解决 _______________
对吗？"
```

### 1.3 澄清问题

```
核心问题：
- "你能举一个具体的例子吗？"
- "这个问题你自己遇到过吗？"
- "现在你是怎么解决这个问题的？"

动机问题：
- "为什么这个问题对你很重要？"
- "如果不解决会怎样？"

愿景问题：
- "理想情况下，解决方案是什么样的？"
- "你最希望它能做到什么？"
```

## 第二步：挖掘真实需求

### 2.1 表面需求 vs 真实需求

```
用户说的（表面）          用户真正想要的（深层）
─────────────────────────────────────────────
"我想做一个笔记应用"    → 更好地组织和找到信息
"我想做一个社交平台"    → 找到志同道合的人
"我想做一个效率工具"    → 减少重复劳动，节省时间
```

### 2.2 五个为什么

```
问题：_______________

为什么这是个问题？→ _______________
为什么会这样？→ _______________
为什么重要？→ _______________
为什么现有方案不行？→ _______________
为什么你想解决它？→ _______________
```

### 2.3 用户画像

```
谁会用这个产品？

主要用户：_______________
- 他们是谁？（年龄、职业、背景）
- 他们的日常是什么样的？
- 他们在什么场景下会遇到这个问题？
- 他们现在怎么解决？
- 他们愿意为解决方案付费吗？

次要用户：_______________
- 还有谁可能会用？
```

## 第三步：发散探索

### 3.1 场景发散

```
核心场景：_______________

相关场景：
- 之前会发生什么？
- 之后会发生什么？
- 有没有类似的场景？
- 极端情况是什么？
```

### 3.2 解决方案发散

```
用户想要的方案：_______________

其他可能的方案：
- 如果用完全不同的方式呢？
- 如果简化到极致呢？
- 如果做得更复杂/完整呢？
- 如果结合其他产品呢？
- 如果不做产品，用服务解决呢？
```

### 3.3 "如果...会怎样？"

```
- 如果用户量是现在的 100 倍？
- 如果完全免费？
- 如果只能有一个功能？
- 如果目标用户完全不同？
- 如果竞争对手明天就做了？
```

## 第四步：挑战假设

### 4.1 识别隐含假设

```
用户的假设：
- [ ] 这个问题真的存在
- [ ] 用户愿意用新产品
- [ ] 用户愿意付费
- [ ] 技术上可以实现
- [ ] 我们能做得比现有方案好
- [ ] 市场足够大
```

### 4.2 验证假设

```
假设：_______________
验证方式：_______________
结果：[ ] 成立  [ ] 不成立  [ ] 需要调研
```

### 4.3 魔鬼代言人

```
故意唱反调：
- "如果这个问题根本不重要呢？"
- "如果用户不愿意改变习惯呢？"
- "如果大公司明天就做了呢？"
- "如果技术上做不到呢？"
```

## 第五步：初步收敛

### 5.1 核心洞察

```
经过探索，我们发现：

核心问题是：_______________
目标用户是：_______________
关键场景是：_______________
可能的解决方向：_______________
```

### 5.2 待验证问题

```
还需要调研/验证的：
- [ ] _______________
- [ ] _______________
- [ ] _______________
```

### 5.3 下一步

```
[ ] 需要市场调研 → 进入 market-research 流程
[ ] 概念已清晰 → 进入 product-definition 流程
[ ] 需要继续探索 → 继续对话
```

## 对话技巧

### 好的提问

```
✅ 开放式问题："你能多说一点吗？"
✅ 具体化问题："能举个例子吗？"
✅ 假设性问题："如果...会怎样？"
✅ 对比问题："和 X 相比有什么不同？"
✅ 感受问题："这让你感觉怎么样？"
```

### 避免的提问

```
❌ 引导性问题："你是不是觉得...？"
❌ 封闭式问题："是不是？"
❌ 多重问题：一次问太多
❌ 评判性问题："你为什么会这么想？"
```

### 积极倾听

```
- 复述确认："你的意思是...？"
- 情绪反馈："听起来这让你很困扰"
- 追问细节："能具体说说吗？"
- 总结归纳："所以核心是...？"
```
