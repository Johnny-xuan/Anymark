 AnyMark - 收藏自由 - 开源免费版

100% 免费开源的 AI 智能书签管理 Chrome 扩展

## 特性

- **AI 智能分析** - 配置自己的 API Key（豆包/OpenAI/Claude）
- **本地分析** - 无需 API 也可使用基础分类
- **键盘导航** - j/k/Enter/Space/? 快捷操作
- **Chrome 同步** - 实时同步原生书签
- **Pixel Buddy** - 可爱的桌面助手，5种主题随心换
- **无需注册** - 完全本地化，保护隐私

## 快速开始

```bash
cd smart-bookmarks
npm install
npm run build
```

然后在 Chrome 中加载 `smart-bookmarks/dist/` 目录。

## 配置 AI（可选）

1. 打开扩展侧边栏
2. 进入设置 → AI 设置
3. 输入你的 API Key
4. 保存即可使用 AI 分析

## Pixel Buddy 主题

5 种主题风格：
- 🔵 经典像素 - 清爽专业
- 🟣 赛博霓虹 - 未来科技
- 🟢 森系萌芽 - 自然治愈
- 🟠 余晖探索 - 活力探险
- ⚫ 极客终端 - 黑客风格

**使用方法：**
1. 打开侧边栏 → 设置 → 界面设置
2. 选择 Pixel 小助手主题
3. 点击"应用主题"按钮刷新所有页面
4. 在任意网页点击 Pixel Buddy 打开侧边栏

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+B` | 打开侧边栏 |
| `Ctrl+J` | 快速收藏 |
| `j/k` 或 `↑/↓` | 上下移动 |
| `Enter` | 打开书签 |
| `/` | 搜索 |
| `?` | 帮助 |
