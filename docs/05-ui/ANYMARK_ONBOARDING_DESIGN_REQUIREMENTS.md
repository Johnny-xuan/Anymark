- **产品名称**: AnyMark - 收藏自由

1. **建立品牌认知**: 传达"AnyMark - 收藏自由"品牌和"收藏自由"理念

- **主Logo**: AnyMark - 收藏自由文字 + 书签图标

### 字体规范
- **中文**: 苹方/PingFang SC、思源黑体

- **正文**: Regular/Medium

## 🎭 设计风格指导

### 视觉风格
- **现代简约**: 干净的布局，充足的留白
- **科技感**: 渐变、阴影、流畅动画
- **亲和力**: Koda助手形象增加温暖感
- **专业感**: 保持品牌调性

### 插画风格
- **像素风格**: Koda助手采用像素/复古游戏风格
- **简约线条**: 功能图标使用线性设计
- **色彩搭配**: 品牌色为主，适当使用暖色点缀

### 动画效果
- **滚动动画**: Section切换的流畅过渡
- **悬停效果**: 按钮和卡片的微交互
- **加载动画**: Koda思考/分析的状态动画
- **页面转场**: 平滑的视觉过渡

## 🖼️ 关键设计元素

### Koda助手形象
- **设计风格**: 8-bit像素风格或简约扁平化
- **表情变化**: 思考、帮助、推荐等不同状态
- **互动元素**: 可以设计简单的对话气泡
- **色彩应用**: 使用品牌渐变色

### 功能展示
- **AI分析可视化**: 展示Koda分析网页的过程
- **分类演示**: 书签自动归类的动效
- **快捷键可视化**: 键盘按键的视觉呈现

### 图标系统
- **搜索图标**: 放大镜 + AI符号
- **收藏图标**: 书签 + 星星
- **分类图标**: 文件夹 + AI标签
- **助手图标**: Koda头像 + 对话气泡

## 📱 响应式要求

### 桌面端 (1200px+)
- 全宽展示，充分利用屏幕空间
- 左右分栏布局（如适用）
- 悬停效果和微交互

### 平板端 (768px-1199px)
- 自适应布局，保持可读性
- 触摸友好的交互元素
- 简化动画效果

### 移动端 (<768px)
- 单列布局
- 大按钮设计
- 简化的导航
- 移动端特有的手势交互
